23 // A math object is a strange thing:
24 // a math_elt object has 4 fields, one is a pointer to another object.
25 // Other 3 objects are : cmd, chr, token in some cases.
26 // it can be cmd, chr, font (if cmd==12 or cmd==11), and font is one of
27 // math_f_it,math_f_rm,math_f_cal,math_f_bb,math_f_sf,math_f_tt,math_f_bf,
28 // it can be math_list_cmd, pos, type, where pos is a position into a
29 // table of math lists, and type is one of
30 // math_open_cd, math_dollar_cd, math_ddollar_cd,math_env_cd, math_LR_cd,
31 // math_hbox_cd,math_argument_cd,
32 // it can be math_xml_cmd, pos, type, where pos is a position into a
33 // table of XML elements.
