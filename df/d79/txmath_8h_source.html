26 // A math object is a strange thing:
27 // a math_elt object has 4 fields, one is a pointer to another object.
28 // Other 3 objects are : cmd, chr, token in some cases.
29 // it can be cmd, chr, font (if cmd==12 or cmd==11), and font is one of
30 // math_f_it,math_f_rm,math_f_cal,math_f_bb,math_f_sf,math_f_tt,math_f_bf,
31 // it can be math_list_cmd, pos, type, where pos is a position into a
32 // table of math lists, and type is one of
33 // math_open_cd, math_dollar_cd, math_ddollar_cd,math_env_cd, math_LR_cd,
34 // math_hbox_cd,math_argument_cd,
35 // it can be math_xml_cmd, pos, type, where pos is a position into a
36 // table of XML elements.
