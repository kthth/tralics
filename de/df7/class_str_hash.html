Detailed Description

The string_hash table.

It is like the hashtable below (Text and Next) but it is extensible (maybe, we should make the hashtable extensible) Each item has a pointer to a LabelInfo element. If the pointer is not null we consider the slot of foo, that has a \label{foo} or \ref{foo} At bootstrap, we fill the table with true, false, spacebefore, center,... This uses utf8 encoding. The token can be dumped via its Value which can be ascii, utf8 or latin1 (XML syntax)
